# AI Agent Pipeline

Этот репозиторий содержит пайплайн для работы с ИИ-агентами, ориентированными на анализ данных, очистку, создание признаков, моделирование и отчетность. Пайплайн использует Langchain, MCP адаптеры и Jupyter для автоматизации различных задач на различных этапах машинного обучения.

## Обзор проекта

Проект включает набор агентов, которые выполняют следующие задачи:

1. Exploratory Data Analysis (EDA)
Анализирует структуру датасета, находит пропуски, дубликаты и другие проблемы качества данных.
2. Очистка данных
Очищает датасет, устраняя конкретные проблемы, выявленные на этапе EDA.
3. Создание признаков (Feature Engineering)
Создает новые значимые признаки на основе очищенного датасета.
4. Моделирование
Предлагает подходящие модели машинного обучения и метрики для решения задачи.
5. Обучение моделей
Обучает модели на датасете, оценивает их производительность с использованием различных метрик и выбирает лучшую модель.
6. Отчетность
Генерирует финальный отчет, резюмируя этапы и результаты пайплайна.

## Требования
- Python 3.7+
- Jupyter Notebook
- Необходимые Python пакеты:
  - langchain
  - langchain_mcp_adapters
  - pydantic
  - dotenv
  - joblib
  - numpy
  - pandas

## Настройка

1. Клонируйте репозиторий:
```
git clone https://github.com/Anton-gra/ml-agent-pipeline.git
cd ml-agent-pipeline
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Настройте переменные окружения, создав файл .env и добавив:
```
OPENROUTER_API_KEY=<ваш_openrouter_api_key>
JUPYTER_URL=<ваш_jupyter_url>
JUPYTER_TOKEN=<ваш_jupyter_token>
```
## Запуск пайплайна
Чтобы запустить пайплайн, выполните следующую команду в терминале:
```
python pipeline.py
```
Пайплайн выполнит следующие шаги:

1. Запросит путь к директории с датасетом.
2. Выполнит этапы (EDA, очистка, создание признаков, моделирование, обучение и отчетность).
3. Выведет финальные результаты и отчет.
### Агенты
- EDA-агент: Выполняет Exploratory Data Analysis, выявляет проблемы в датасете и генерирует отчет.
- Агент очистки: Устраняет конкретные проблемы, выявленные EDA-агентом.
- Агент создания признаков: Обогащает датасет новыми признаками на основе очищенных данных.
- Агент моделирования: Предлагает модели и метрики для оценки на основе датасета.
- Агент обучения: Обучает выбранные модели, оценивает их и сохраняет лучшую модель.
- Агент отчетности: Генерирует финальный отчет на основе результатов пайплайна.

**Примечания**
Пайплайн разработан для работы в среде Jupyter и использует комбинацию Python, Pandas и инструментов Langchain для выполнения задач.
Вы можете изменять путь к датасету и другие конфигурации в соответствии с вашими потребностями.
